Exemplo Web Services REST

Objetivos

  • Explorar a aplicação (tanto código-fonte quanto através de navegação) para ver as funcionalidades e o comportamento delas
  • Reimplementar funcionalidades principais como Webservices REST
  • Recriar as telas usando HTML + JS + AJAX para consumir dados do Webservice

Informações importantes

  • Este exemplo foi baseado no mesmo exemplo usado para apresentação do JPA e cria um banco de dados H2 e insere alguns dados na primeira execução.
  • **** IMPORTANTE **** Após abrir o projeto no Netebeans, rodar o comando Clean and Build ANTES de executá-lo.
  • Para apresentar as imagens, obtê-las a partir do seguinte endereço: https://1drv.ms/u/s!AnEDdBYgtN7hg6ZL_cMuH0LA7bpsFQ?e=fVyrpF.
    • Mover o arquivo .zip para um diretório adequado da sua máquina e descompactar (OBS: EVITAR COLOCAR DENTRO DA PASTA DO PROJETO)
    • Após descompactar, alterar o diretório onde elas se encontram no arquivo /src/main/resources/application.properties na propriedade app.imagem-path e reiniciar a aplicação.
    • Todas elas foram geradas através do uso da API Fake Face (https://hankhank10.github.io/fakeface/)
